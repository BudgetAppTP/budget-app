\section*{Úvod}
\addcontentsline{toc}{section}{Úvod}

Súčasné rodiny čoraz viac využívajú softvér na sledovanie príjmov, výdavkov a úspor. \textbf{Budget Tracker} je webová aplikácia pre osobné financie vytvorená s použitím Python a Flask. Pomáha používateľom zaznamenávať príjmy a výdavky, plánovať mesačné rozpočty, sledovať finančné ciele a importovať účtenky prostredníctvom QR kódov kompatibilných so slovenským elektronickým pokladničným systémom \texttt{eKasa}.
